
Project Reminder app
## Download the code

git clone -b master https://github.com/dagamo/reminder_app.git

## Install dependences

npm install

## run the project
npm start

## Functionality
1) Create a Reminder.
2) Update the reminder.
3) Delete a reminder.
4) List the reminder of a day.
5) A calendar dinamicly

## See the reminders.
1) Click on a specific day of the calendar.
